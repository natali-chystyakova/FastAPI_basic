# FastAPI_basic

## Project Description
**FastAPI_basic** is a basic template project that will help you get started with FastAPI.

## Tech Stack
- Python 3.11,
- FastAPI,
- Uvicorn (ASGI server),
- Pydantic ,
- aiosqlite,
- SQLAlchemy,
- pytest,
- Docker,  etc.
